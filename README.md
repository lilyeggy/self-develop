# Spirit - 个人成长Agent系统

一个支持即时思考记录、思路展开、复盘回顾和认知提升的智能个人成长助手。

## 核心功能

### 1. 多渠道输入
- **Web界面**: 轻量级响应式设计，支持快速输入
- **API接口**: 开放API，支持第三方集成
- **笔记插件**: Notion、Obsidian、印象笔记等插件

### 2. 核心Agent模块
- **输入处理Agent**: 接收、解析、初步处理用户输入
- **内容组织Agent**: 自动分类、关联、结构化存储
- **分析Agent**: 认知模式识别、成长建议生成
- **提醒Agent**: 定期回顾、思考引导

### 3. 智能分析
- 内容自动分类(想法/问题/感悟/计划等)
- 思路展开与问题延伸
- 周期性回顾总结
- 认知变化追踪

### 4. 隐私保护
- 端到端加密存储
- 本地部署选项
- 数据导出支持

## 技术栈

- **后端**: Python 3.11+ / FastAPI
- **数据库**: PostgreSQL + Redis
- **前端**: React + TypeScript
- **AI/NLP**: OpenAI API
- **加密**: AES-256-GCM

## 快速开始

### 本地部署

```bash
# 1. 克隆项目
git clone https://github.com/yourusername/spirit.git
cd spirit

# 2. 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或 venv\Scripts\activate  # Windows

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置环境变量
cp .env.example .env
# 编辑 .env 文件

# 5. 初始化数据库
python -m spirit.db.init

# 6. 启动服务
python -m spirit.main
```

### Docker部署

```bash
docker-compose up -d
```
